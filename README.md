# Digital Clock Web App

This web application displays a digital clock with a sleek design. It features real-time updating of hours, minutes, and seconds. The clock's appearance and animation add visual appeal.

## Table of Contents

- [Features](#features)
- [Prerequisites](#prerequisites)
- [Usage](#usage)
- [Files](#files)
- [Media Queries](#media-queries)

## Features

- Real-time display of hours, minutes, and seconds.
- Responsive design using CSS media queries.
- An animated background and clock border.

## Prerequisites

- Modern web browser (Chrome, Firefox, Safari, etc.)

## Usage

1. Clone or download this repository.
2. Open `index.html` in your web browser.

The digital clock will be displayed in the center of the page. The font size and layout adjust based on the screen width.

## Files

- `index.html`: Contains the HTML structure for the digital clock.
- `style.css`: Defines the visual styling and layout of the digital clock.
- `clock.js`: Provides the JavaScript logic to update the clock every second.

## Media Queries

The clock's responsiveness is achieved using CSS media queries. The clock's font size and layout adapt based on the screen width.

- For screens with a maximum width of 443px, the font size increases to improve readability. The clock's layout adjusts to fit the screen width while maintaining a pleasing appearance.

Feel free to explore and modify the CSS media queries in `style.css` to customize the responsiveness according to your preferences.
